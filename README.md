# Single-Page-Application-Summative-Lab-3

## Dictionary Search Tool

A simple dictionary search web app that uses the free Dictionary API to fetch and display definitions, phonetic spelling, audio pronunciation, and synonyms.

---

## Features

- Search for any English word  
- Displays:
  - Word
  - Phonetic spelling
  - Audio pronunciation (if available)
  - Definitions
  - Synonyms (if available)
- Graceful handling of:
  - Missing data (no phonetics, no synonyms, etc.)
  - Invalid words
  - Empty searches
- Clean and responsive UI

---

## Technologies Used

- **HTML**
- **CSS**
- **JavaScript**
- **Fetch API**
- **Dictionary API**  
  (https://api.dictionaryapi.dev)
